# Copy of CRAN maptools

From the archived project : https://maptools.r-forge.r-project.org/index.html  

---  

This project has had fixes in src/pip.c

It now builds in R 4.5.2 with this:  
```
R -e "devtools::install_url('https://github.com/rachelicr/r-maptools/archive/refs/heads/main.zip')"

```
